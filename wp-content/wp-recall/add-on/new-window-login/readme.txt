== Установка/обновление ==

Устанавливается как обычное дополнение плагина WP-Recall без каких либо дополнительных загрузок файлов на сервер. В консоли сайта выбираем «WP-RECALL» -&gt; «Дополнения» -&gt; «Выбрать файл» -&gt; «Установить» -&gt; активировать дополнение.

Поддержка автоматических обновлений плагина WP-Recall, смотрим <a href="https://codeseller.ru/avtomaticheskie-obnovleniya-dopolnenij-plagina-wp-recall/" target="_blank">ТУТ</a>

== Настройка ==

По мере необходимости вы можете произвести настройки установленного вами дополнения в «WP-RECALL» -&gt; «НАСТРОЙКИ» выбрав вкладку настроек дополнения «New Window Login».
